## 拾取判定

将实体id代替实体颜色走一遍渲染流程，将渲染结果回传cpu，根据鼠标所在像素的颜色得出所选中的实体

对所有实体进行渲染：
所有实体的mesh作为输入，可能可以利用`SceneRenderer::sub_mesh_render_elements_`
```cpp
struct SubMeshRenderElement  
{  
    SubMeshRenderData* sub_mesh;  
    MaterialRenderData* material;  
    const Mat4* matrix = nullptr;  
    const Mat4* pre_frame_matrix = nullptr;  
    const std::vector<Mat4>* instanced_matrices;  
    const Bounds* world_bounds;  
    bool active = true;  
    bool cast_shadow = true;  
};
struct SubMeshRenderData
{
	// 获取唯一标识符  
	uintptr_t get_id() const;  
	  
	// 绑定子模型的顶点缓存，调用渲染指令  
	uint32_t draw(Backend::CommandBuffer& command_buffer, const Backend::PipelineLayout& pipeline_layout);
}
```
渲染实现参考SceneRender::geometry_pass_，可能需要一个单独的渲染图
